# Alexa Clauss Personal Website

A modern, minimalistic personal website with bright colors and a sleek design.

## Features

- **Homepage** with personal bio and photo placeholder
- **Portfolio** page to showcase your work
- **Blog** page for your thoughts and updates
- **Resume** page for work experience and skills
- Responsive design that works on all devices
- Modern navigation with active state indicators
- Curvy, friendly font (Comfortaa)
- Bright, minimalistic color scheme

## Color Scheme

- Primary: Pink (#FF6B9D)
- Secondary: Teal (#4ECDC4)
- Accent: Yellow (#FFE66D)
- Text: Dark gray (#2C3E50)

## Getting Started

1. Open `index.html` in your web browser to view the website
2. Customize the content in each HTML file
3. Replace the image placeholder with your actual photo
4. Add your portfolio projects, blog posts, and resume information

## File Structure

```
website/
├── index.html      # Homepage
├── portfolio.html  # Portfolio page
├── blog.html       # Blog page
├── resume.html     # Resume/Work Experience page
├── styles.css      # All styling
└── README.md       # This file
```

## Customization

### Adding Your Photo

Replace the placeholder div in `index.html` with an actual image:

```html
<img src="path/to/your/photo.jpg" alt="Alexa Clauss" class="hero-image">
```

### Updating Colors

Modify the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #FF6B9D;
    --secondary-color: #4ECDC4;
    --accent-color: #FFE66D;
}
```

### Adding Content

Simply edit the HTML files to add your personal information, projects, blog posts, and work experience.

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
